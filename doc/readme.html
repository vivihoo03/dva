使用的技术：

底层基于Reactjs、redux、redux-saga、react-router等
打包使用webpack
使用babel转码es6到es5
样式预处理使用sass 。

样式命名：

统一使用下划线，公共组件class以ui_开头，后面跟上组件的名字，组件相关的样式包在命名内部。
例如：公共：.ui_slider:

  1. .ui_slider{
  2.     img{
  3.     }
  4. }

业务组件以class名称以.uib开头，后面跟上组件的名字，组件相关的样式包在命名内部。
.uib_listview:

  1. .uib_listview{ 
  2.     .title{
  4.     } 
  5. }

公共base.1.0.0.scss和var.1.0.0.scss
base.1.0.0.scss基于normalize.css,
var.1.0.0.scss提供基础变量，和UI标注一一对应。
比如UI给出的body的标注是list-primary,则可以这么写：

  1. @import '../../../../styles/var.1.0.0.scss';
  2. body {
  3.  color:$primary;
  4.  font-size:$list;
  5. }

其中的变量名已经在var.1.0.0.scss里面声明好了。

JS组件命名：

组件名统一使用驼峰命名：
公共: imgPreview、listView
业务组件：bListView、bImgPreview
